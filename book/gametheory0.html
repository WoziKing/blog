# 博弈论学习笔记

# 0 前言

  课程来源于Bilibili，浙大博弈论基础，视频源：https://www.bilibili.com/video/BV1AK4y1S7Au

# 1 概述

# 1.1 什么是“博弈”

  “博”：博览全局；
  “弈”：对弈棋局。
  从而谋定而动，而非轻举妄动。

  博弈是指在一定的游戏规则的约束下，基于直接相互作用的环境条件，各参与人依据所掌握的信息，选择各自的策略（行动），以实现利益最大化的过程。

  注：

  • 直接相互作用：一人在做决策时要考虑对另一人造成的影响，以及这种影响反过来对该人的影响。
  • 信息：信息是不对称的。
  • 利益最大化是目标。
例1：打猎遇狮

两个人同行打猎，遇到狮子，一人卸下重物狂奔，同伴不解：“汝能胜狮？”答曰：“非需胜狮，只需胜汝。”

行为选择跟对手有关，还跟外部环境有关：

  • 要搞清竞争对手是谁，以及未来的对手可能是谁：不是和狮子比，而是和同伴比。
  • 差异化战略：若碰到非常聪明狡猾的狮子，谁跑得快谁先被吃，此时两人要往相反的方向跑。
  • 合作关系：若碰到比较弱小的狮子，直接将狮子打死。

  博弈既可以是竞争，又可以是合作。

例2：数字游戏

每个同学写一个1至100之间的自然数（整数，包括1与100），然后求出所有数字的平均数，如果你的数字最接近这个平均数的二分之一，那么你将胜出。

  答案是不固定的，关键在于需要思考别人会写什么数字，如果别人写50，自己将写25；自己写25，别人也有可能写25，所以在自己将写12……以此循环，最小的则是1。
  然而，写1并不能胜出，因为只有所有人都写1时，才能胜出，这种情况是基本不可能的。
  如果是乱写的，平均数将接近50，然而在规则限制的前提下，平均数将小于50。

  博弈必须学会换位思考。
  博弈，只需领先一步，高人一筹。

  博弈的核心在于整体思维基础上的理性换位思考，用他人的得益去推测他人的策略，从而选择最有利与自己的策略。

例3：罗森塞蜈蚣博弈

在两个人面前放一个钱盒，假如一开始放50块钱，如果第一个人结束，则第一个人获得40块（4:1）。如果第一个人继续，钱盒中的钱成倍，而比例对调，即若第二个人结束，则第一个人获得20，第二个人获得80。以此循环。当钱盒装满时（800块），游戏自动结束，第一个人获得640块，第二个人获得160块。
